GitLab - Test Challenge
====

Write a small Ruby based script/app that will get HTTP response times over 5 minutes from your location to https://gitlab.com

Prerequisites
----

- Ruby ~> 2.0

Steps
----

Execute the method from the command line with an URL as parameter

Run the program like this:

```
ruby response_time.rb https://gitlab.com
```

It will return the HTTP Response Times over 5 minutes.
